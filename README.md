# South Tyrol Music Network

An interactive website that visualizes the South Tyrol music scene through an interactive node graph.

## Features

- **Interactive Graph**: Canvas-based visualization with draggable view and zoom
- **Producer Information**: Click on producer nodes to view detailed information in modal windows
- **Mobile Responsive**: Touch-friendly controls for mobile devices
- **Real-time Data**: Fetches producer data directly from Google Sheets
